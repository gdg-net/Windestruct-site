Windows FU

Introduction

Windows FU is an another Windows modification of mine that resembles a parody operating system. This time it's Windows FU.

Why?

After I completed Windows RG, I felt like making Windows FU next. Windows FU is a successor to Windows RG. This is an another favor to the Internet.

How?

I began by installing vanilla Windows XP to VMWare (this time I could install drivers because the enitre disk wasn't imaged). For programs, I used a VM with Visual Studio 2005 Express and copied the files over to my Windows installation. After a lot of time, all the programs were done.

Next, I created a theme and added some custom wallpapers. After I was finished, I began creating a custom installation screen with Resource Hacker.

I used nLite to integrate all the data and after painstakingly testing everything the image was finished.
